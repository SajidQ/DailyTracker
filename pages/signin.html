DailyTracker

Welcome to the DailyTracker! This is a website that uses the Google Calendar API to help you keep track of how you spend your day along with planning goals for the day, month, and year.

This is an app that was created as a side project for experience and personal use. If you like the app, feel free to use it or at least try it out! Any information you type is sent straight to your Google Calendar account meaning no information about you is saved on this site. You can check out the code on github. Check out the images of the app below and click Authorize when you're ready to get started.
